# 🚀 ServiceNow REST API Integration

## 📌 Overview

This project demonstrates how to integrate ServiceNow with an external system using Outbound REST APIs.

The application connects to an IP Geolocation API and fetches real-time location data based on IP address.

## 💡 Objective

To understand and implement real-world integrations by connecting ServiceNow with external APIs and processing JSON responses.

## ⚙️ Features

- Configured REST Message (GET method)  
- Used endpoint and query parameters (API key)  
- Handled JSON response from external API  
- Extracted useful fields:
  - 🌍 Country  
  - 🏙️ City  
  - ⏰ Time Zone  
  - 💰 Currency  
- Tested API using HTTP methods and status codes  

## 🔁 Workflow

1. Send request to external API  
2. Receive JSON response  
3. Parse response data  
4. Display required fields  

## 📈 Outcome

- Gained hands-on experience with ServiceNow integrations  
- Improved understanding of APIs and JSON handling  
- Learned how real systems communicate  

## 🛠️ Technologies Used

- ServiceNow REST Message  
- JSON Parsing  
- HTTP Methods
